# Los Alpes - Muebleria

Repo base para separar la aplicacion web Next.js del backend Express con TypeScript.

## Estructura

```txt
/
├── frontend/  # Proyecto Next.js actual
└── backend/   # API Express + TypeScript
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
npm install
npm run dev
```

La API expone inicialmente:

```txt
GET /health
```
